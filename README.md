# 完全匿名目安箱 - 高等学校用

高等学校向けの完全匿名意見投稿システムです。生徒が安心して意見や提案を投稿できるプラットフォームを提供します。

## 🌟 特徴

- **完全匿名性**: 投稿者の特定は技術的に不可能です
- **簡単操作**: シンプルで分かりやすいインターフェース
- **カテゴリー分類**: 意見を種類別に整理
- **モダンデザイン**: レスポンシブ対応で、スマートフォンでも快適に利用可能
- **リアルタイム表示**: 投稿された意見をすぐに閲覧可能

## 📋 機能

### 投稿機能
- カテゴリー選択（学習、施設、イベント、人間関係、その他）
- 最大500文字のメッセージ入力
- リアルタイム文字数カウント
- 投稿完了通知

### 表示機能
- 投稿一覧表示
- カテゴリー別の色分け
- 投稿時間の表示（相対時間）
- 総投稿数の統計

### 匿名性保証
- IPアドレスの記録なし
- Cookie・セッション情報の収集なし
- ユーザーエージェント情報の保存なし
- 位置情報の取得なし

## 🚀 使い方

### 基本的な使い方
1. ブラウザで `index.html` を開く
2. カテゴリーを選択
3. メッセージを入力（最大500文字）
4. 「匿名で投稿する」ボタンをクリック
5. 投稿完了！

### GitHub Pages でのデプロイ
1. このリポジトリをフォーク
2. Settings > Pages で GitHub Pages を有効化
3. Source を `main` ブランチに設定
4. 公開URLにアクセス

## ⚠️ 注意事項

### 現在の実装について
現在、投稿データは**ブラウザのLocalStorage**に保存されています。これは以下を意味します：
- データはブラウザごとに独立して保存されます
- 他のユーザーと投稿を共有することはできません
- ブラウザのキャッシュをクリアするとデータが消えます

### 本番環境での推奨事項
学校で実際に運用する場合は、以下の実装を推奨します：

1. **バックエンドサーバーの構築**
   - Node.js + Express
   - Python + Flask/Django
   - PHP などのサーバーサイド言語

2. **データベースの使用**
   - MongoDB
   - PostgreSQL
   - MySQL など

3. **匿名性の強化**
   - IPアドレスの記録禁止設定
   - HTTPS通信の必須化
   - プライバシーポリシーの明示

4. **モデレーション機能**
   - 不適切な投稿の削除機能
   - 管理者用ダッシュボード

## 🔒 セキュリティ

- XSS対策: すべてのユーザー入力をエスケープ処理
- 文字数制限: 500文字までの制限
- バリデーション: 必須項目のチェック
